# Contributing

This is a portfolio demonstration, but focused improvements are welcome.

## Development flow

1. Create a branch from `main`.
2. Keep changes small and purposeful.
3. Run `npm run typecheck`, `npm run lint` and `npm run build`.
4. Include screenshots for visible interface changes.
5. Explain accessibility and mobile impacts in the pull request.

## Commit examples

```text
feat: add product shade selection
fix: preserve cart variants in local storage
perf: defer hero scene until client render
docs: clarify portfolio limitations
```

Do not commit secrets, copied retailer assets, customer information or unlicensed product models.
